# NTIRE 2026 Mobile Real-World Image SR — team12_SNOWVision

## Setup

```bash
git clone https://github.com/ironblog/NTIRE2026_Mobile_RealWorld_ImageSR.git
cd NTIRE2026_Mobile_RealWorld_ImageSR

pip install torch torchvision numpy pillow
pip install pyiqa  # for eval.py only
```

## Directory Structure

```
models/team12_SNOWVision/
    __init__.py
    model.py                  # Mobile-HGSR (1.13M params, x4 SR)
model_zoo/team12_SNOWVision/
    best.pth                  # EMA weights (4.67 MB)
data/test/
    lr/                       # input LR images (place here before running)
    sr/                       # output (generated by test.py)
test.py                       # inference script
eval.py                       # official evaluation (unchanged)
```

## Run Inference

```bash
python test.py
```

Reads from `data/test/lr/`, writes x4 SR to `data/test/sr/`.

## Run Evaluation

```bash
python eval.py --output_folder data/test/sr --target_folder data/test/hr --metrics_save_path results/ --gpu_ids 0
```

## Model

- **Architecture:** Mobile-HGSR (Hybrid-Group SR with high-frequency bias and gated refinement)
- **Parameters:** 1,132,887 (1.13M)
- **Scale:** x4
- **Tile:** 128x128 LR, 24px overlap, Hann blending
- **Latency:** <166 ms on Dimensity 8400 (NNAPI FP16)
